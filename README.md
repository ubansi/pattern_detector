Pattern Detector
--
このプログラムは文字列配列のパターンを検知し、正規表現形式で表現します。(WIP)

# 例

## 利用例
### 通常モード
入力を以下とした場合

```javascript
[
    "aa-01",
    "aa-02",
    "ab-03",
]
```
以下のような結果を出力する。

```javascript
'a[a-z]-[0-9]{2}'
```

文字列のパターンを
* 数値 `[0-9]`
* 小文字アルファベット `[a-z]`
* 大文字アルファベット `[A-Z]`
* その他記号 `[.,+-*]`など

に分類し、妥当なパターンに分割する。

### 厳密モード
厳密モードでは以下のように解釈する。

```javascript
'a[a-b]-0[0-3]'
```

パターン分けの他に出てこない文字列を許容しない形式で結果を返す。

## インストール・ビルド方法
## 環境構築
`docker-compose build --no-cache`

### コンテナ起動
`docker-compose up -d`

### コンテナ内に入る
`docker-compose run -it pd-node bash`
